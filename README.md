# Stable Diffusion LoRA — NIH Chest X-ray Domain Adaptation

Fine-tuning Stable Diffusion v1.5 on the NIH ChestX-ray14 dataset (113K images)
using Low-Rank Adaptation (LoRA) for medical image domain adaptation.

## Problem

General-purpose text-to-image models like Stable Diffusion are trained on
natural images and generate colorful anatomical illustrations when prompted
with medical concepts — not realistic radiographs. This project adapts the
model toward chest X-ray appearance using a lightweight fine-tuning technique.

## Method

| Component | Detail |
|-----------|--------|
| Base model | `runwayml/stable-diffusion-v1-5` |
| Adaptation | LoRA injected into UNet attention layers (to_q, to_k, to_v, to_out) |
| LoRA rank | 8 |
| Alpha | 16 |
| Trainable parameters | ~1% of total weights |
| Training steps | 2,500 |
| Optimizer | AdamW, lr = 5e-5 |
| Dataset | NIH ChestX-ray14 — 113,694 images |

Text conditioning prompt template:
```
"chest X-ray, frontal PA view, high resolution, [Finding]"
```

## Results

| Metric | Value |
|--------|-------|
| Best SSIM | 0.4185 |
| Final training loss (MSE) | 0.0737 |
| Training time | 97 minutes (single GPU) |

The LoRA-adapted model produces grayscale radiograph-like outputs compared
to the colorful anatomical illustrations generated by the base model.
SSIM peaks at an intermediate checkpoint (~step 1000) and does not
monotonically increase — consistent with typical diffusion fine-tuning behavior.

## Findings Evaluated
Cardiomegaly · Effusion · Consolidation · Pneumonia · Nodule
